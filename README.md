# Hotel Management System

A Python-based hotel management system that handles guest check-in, check-out, billing, Excel reports, and email automation.

## What it does
- Add and manage hotel guests
- Automatic billing calculation
- Generates Excel reports with openpyxl
- Sends automated emails with smtplib

## Tools used
- Python
- openpyxl
- smtplib

## How to run
1. Install requirements: `pip install openpyxl`
2. Run: `python hotel_system.py`
